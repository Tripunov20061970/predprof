- Вывод первых десяти номеров
## Установка
Установить проект можно с помощью команды
'git clone https://github.com/Tripunov20061970/predprof/'

## Как пользоваться
Чтобы запустить файлы Python, воспользуйтесь командой
'python3 <название файла>'
